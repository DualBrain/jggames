The original concept came from my son Charlie, age 5.
These files are:

Sentinel        Compiled version of the program
Sentinel.b09    Basic09 source code

Sentinel
========
The program only works with the normal 32 X 16 vdg screen.
It is compatible with of OS9 level 1 and 2.  Have fun!
