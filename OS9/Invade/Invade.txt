The three files contained in this archive represent the C sourcecode and compiled version of the space invaders style game "Invasion."These files are:Invade          Compiled version of the programInvade.c        C source codeInvade_hi.scr   High score file.Invasion========The program only works with the normal 32 X 16 vdg screen.It is compatible with of OS9 level 1 and 2.
